# Experiment 2 — 2x2 Factorial Design

## Setup
- 4 conditions: (Free/State) x (Raid/NoRaid)
- 3 seeds per condition, 200 years, 4 bands per condition (all same regime)

## 2x2 Table — Mean cooperation at year 200

|             | Raiding ON | Raiding OFF |
|-------------|-----------|------------|
| **Free**    | 0.506   | 0.482    |
| **State**   | 0.469   | 0.484    |

## Key metrics

- Free-State divergence WITH raiding: +0.037
- Free-State divergence WITHOUT raiding: -0.002
- Interaction effect (raiding x institutions): +0.039

## Interpretation
- Interaction effect is positive (+0.039): raiding amplifies institutional effects.
- This suggests between-group selection IS causally involved.
