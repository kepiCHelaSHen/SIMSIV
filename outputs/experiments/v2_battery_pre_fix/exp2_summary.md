# Experiment 2 — 2x2 Factorial Design

## Setup
- 4 conditions: (Free/State) x (Raid/NoRaid)
- 3 seeds per condition, 200 years, 4 bands per condition (all same regime)

## 2x2 Table — Mean cooperation at year 200

|             | Raiding ON | Raiding OFF |
|-------------|-----------|------------|
| **Free**    | 0.466   | 0.468    |
| **State**   | 0.491   | 0.499    |

## Key metrics

- Free-State divergence WITH raiding: -0.026
- Free-State divergence WITHOUT raiding: -0.032
- Interaction effect (raiding x institutions): +0.006

## Interpretation
- Interaction effect is near zero (+0.006): raiding does not significantly alter the institutional effect.
- This supports North — institutions alone explain the divergence.
